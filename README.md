## Home24 - Technical Task
 
### REQUIRMENTS TO RUN:
[golang](https://golang.org/)>=1.15.X


### TO RUN THE CRAWLER:
- First clone the repository by running this command on terminal:  `git clone https://github.com/dukagjinramosaj1/home24-task.git`
